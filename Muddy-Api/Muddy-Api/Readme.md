# MuddyApi
This is a web api built on asp.net core 2.1

## Development server

you can run this app through visual studio or equivalent or through docker using 
docker run -p 8080:80 --name myapp Muddy-Api

## Build

docker build -t Muddy-Api .

## Further help

To get more help on the Angular CLI use `ng help` or go check out the [Angular CLI README](https://github.com/angular/angular-cli/blob/master/README.md).
